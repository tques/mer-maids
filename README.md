# M.E.R. Maids

## Project info

**URL**: https://magictrix.itch.io/mer-maids

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How do I play this game
You pilot a M.E.R. Unit. It flies like a plane, but is propelled by water. It refuels in the water and can dive like a sub.
Your goal is to protect 3 cities from bombers and submarines. You fly and shoot with your mouse, left click to jet and right click to fire. 
W boosts the unit while A and D are used to dodge missiles.
